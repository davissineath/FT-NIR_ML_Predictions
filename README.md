# FT-NIR Process Analyzer for Glucose Content Prediction in Powdered Foods
This repository contains the code associated with Sampson Cluff, Emilia Huff, and Davis Sineath's BENG 5910 Final Project. 

### Workflow
1. The data/ folder contains the raw FT-NIR data, data processing code, and processed data.
2. The PLS_model/ folder contains the PLS.ipynb file where we analyzed the data and used partial least squares models to make predictions.
3. Figures used in our final presentation are located in the figure/ folder
